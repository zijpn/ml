### Yeoman scaffolding
```sh
$ sudo npm install -g yo gulp bower generator-gulp-webapp
$ yo gulp-webapp
```

### Build and deploy gh-pages through travis-ci
```sh
$ sudo npm install -g generator-travis-ci
$ yo travis-ci:gh-pages
```
